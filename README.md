## Usage
Run `nightglow help` for help.

## Quick start
1. Run the Nightglow installer with `bash <(curl -Ls https://github.com/ltgcgo/nightglow/releases/latest/download/install.sh)`.
2. Connect to your local SOCKS5 proxy on port 20040.
